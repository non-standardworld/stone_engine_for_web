デモ1: 横書き編集 (LrTb)

クリックしてテキストを編集できます。日本語IME、矢印キー、Backspace/Delete、コピー&ペーストに対応しています。

実装済み機能

  • 日本語IME対応（compositionイベント処理）
  • カーソル移動（矢印キー、Home/End、クリック）
  • テキスト編集（挿入、削除、改行）
  • コピー&ペースト
  • 禁則処理・約物処理との統合

使い方

編集エリアをクリックしてフォーカスしてから、テキストを入力できます。 日本語IMEを使った変換、矢印キーでのカーソル移動、Backspace/Deleteでの削除、 Ctrl+V（またはCmd+V）でのペーストに対応しています。

実装されている機能:
✓ IME統合（compositionstart/compositionend）
✓ カーソル描画（横書き: 縦線、縦書き: 横線）
✓ テキスト挿入・削除
✓ 矢印キーナビゲーション
✓ クリップボード対応
